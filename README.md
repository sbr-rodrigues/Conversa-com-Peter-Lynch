# Conversa-com-Peter-Lynch
Este projeto foi desenvolvido na plataforma NotebookLM (Google), tendo como referência o renomado gestor Peter Lynch, responsável pelo sucesso do Fidelity Magellan Fund e defensor da filosofia “Invista no que você conhece”.
A ferramenta foi programada para simular o estilo de pensamento e comunicação de Peter Lynch, proporcionando ao usuário uma experiência imersiva e sensorial — como se estivesse conversando diretamente com o próprio investidor

Resultados ricos em material criado como:
* Mapa Mental;
* Relatórios financeiros;
* Apresentação;
* Vídeos;
